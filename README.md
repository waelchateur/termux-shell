[+] Installation & Usage

apt update
apt install git -y
git clone https://github.com/waelchateur/termux-shell.git
cd termux-shell
chmod +x *
sh install.sh
exit



or use Single Command

apt update && apt install git -y && git clone https://github.com/waelchateur/termux-shell.git && cd termux-shell && chmod +x * && sh install.sh
